# Assignment 5 - Three.js Scene

## Nalin Verma | naverma@ucsc.edu

This assignment builds an interactive 3D scene using Three.js.

## Features
- Loaded OBJ beetle model
- Textured rotating cubes
- Textured/Colored cylinder
- Floor plane, sky background image, and multiple light sources (Ambient, directional, hemisphere lighting modes)
- Orbit controls and GUI controls for camera/light tuning
